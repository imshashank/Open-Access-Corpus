About Us

The growth of open access (OA) academic publication world- wide challenges libraries and academics about how to organize, rank, and include new work in library catalogs. However, systematic evaluation of trends in cross-disciplinary open access publication is hindered by fragmentation of sources and inconsistent presentation of content. To facilitate research in this space, we have compiled the holdings of the three largest directories of open access academic publications: Directory of Open Access Journals (DOAJ), PubMed Central Open Access (PMC OA), and ArXiv. We standardize record format to support cross-source comparison, and we apply a variety of heuristics and external sources to improve record quality. We publish this corpus of 2,788,559 documents as a freely available resource at http://goo.gl/ KOlgVX.

The OAC Corpus

A corpus of Open Access Articles

The rapid growth of open access (OA) academic publication worldwide challenges academics about how to evaluate this new work and include it in scholarly catalogs. However, a systematic approach to organizing cross-disciplinary open access publication has been hindered by fragmentation of sources and inconsistent presentation of content. To facil- itate research in this domain, we have compiled the holdings of the three largest directories of open access academic publications: Directory of Open Access Journals (DOAJ), PubMed Central Open Access (PMC OA), and arXiv. We standardized the record format to support cross-source comparison, and we applied a variety of heuristics and external sources to improve record quality. We then published this corpus of 2,737,377 documents as a freely available resource at http://goo.gl/KOlgVX.

OAC Front-End

Interacting with the OAC

The main motive behind the project is to be able to categorize and rank all of the Open Access Corpus in categories. Also we want to produce a constant stream of articles much like a “Facebook Wall” with a list of articles in order or their rank inference using machine learning (CO-Author Network) and NLP.

The front end will be a reddit like site. The admin user will be able to see both the back panel and be able to trace back to the original results of clustering and NLP inference. While on front end any user will be able to vote up or down each article and be able to interact with the site.

All Open Access articles can be divided into these sections:

  • Categories (Generalized categories Ex: Science)
  • Tags ( Specialized Topics Ex: Space)
  • Authors
  • Journals

The OAC API

Making the OAC open for all

A WebService/API was created to allow access to the The Open Access Academia Corpus. The WEbService can be accesses using a “GET” HTML call and passing fields like article_id, alternate_id, title etc . The Web Service is built with php and mysql and uses AWS EC2 and AWS RDS for backend.

The WebService can be accessed here : http://api.open-academia.org/. The WebService follows REST protocol and returns a JSON object with the response.

The Web Service has a total of 2020935 articles from 2786115 authors. The total journals are 11854 and total publishers are 4265 . The articles have 914554 unique tags and 1124165 articles have DOI.
